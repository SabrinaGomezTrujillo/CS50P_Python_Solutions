# 🐍 CS50P — Introduction to Programming with Python

My solutions to all problem sets from [CS50's Introduction to Programming with Python](https://cs50.harvard.edu/python/), Harvard University's introductory Python course offered via edX.

Completed: February 2026

---

## 📚 Course Overview

CS50P covers Python from the ground up, with a focus on writing clean, idiomatic code. Each week introduces a new concept through lectures and hands-on problem sets.

---

## 🗂️ Problem Sets

| Week | Topic | Key Concepts |
|------|-------|-------------|
| 0 | Functions & Variables | Functions, arguments, return values, variables |
| 1 | Conditionals | `if`, `elif`, `else`, comparison operators |
| 2 | Loops | `while`, `for`, `break`, `continue` |
| 3 | Exceptions | `try`, `except`, `raise`, input validation |
| 4 | Libraries | `import`, `random`, `statistics`, `sys`, APIs |
| 5 | Unit Tests | `pytest`, test functions, edge cases |
| 6 | File I/O | Reading & writing files, CSV, binary |
| 7 | Regular Expressions | `re` module, pattern matching, validation |
| 8 | Object-Oriented Programming | Classes, methods, properties, `__dunder__` |
| 9 | Et Cetera | `set`, `global`, constants, type hints, unpacking |

---
